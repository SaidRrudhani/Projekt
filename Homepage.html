GPU — The GPU (Graphics Processing Unit) is responsible for drawing images, videos and animations on your screen. It takes over heavy visual work from the CPU so games, 3D applications and video editing run smoothly. A stronger GPU means higher frame rates, better image quality and support for higher resolutions.

CPU — The CPU (Central Processing Unit) is the “brain” of the computer that processes instructions and calculations. It coordinates all other components and decides how quickly programs open and respond. More powerful CPUs can handle multitasking better and speed up tasks like coding, browsing and office work.

RAM — RAM (Random Access Memory) is fast temporary memory that stores data your computer is currently using. More RAM lets you keep more programs and browser tabs open without slowing the system down. It helps the CPU access information quickly so applications feel smooth and responsive.
