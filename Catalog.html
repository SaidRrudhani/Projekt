GPU — The GPU (Graphics Processing Unit) quite like the CPU also handles a type of processing but in the case of the gpu it handles images and video outputs lightening the load on the CPU, higher end GPU's give better performance for different subjects such as:Games,Multitasking heavy applications,etc.

CPU — The CPU (Central Processing Unit) is the “brain” of the computer that processes instructions and calculations. It coordinates all other components and dictates how they each handle certain workloads,for a small size chip the CPU is the main reason why a computer can do any tasks at all.More modern CPU are singeled out by multithreding technology.

RAM — RAM (Random Access Memory) is fast temporary memory that stores data your computer is currently using. The idea of this storage lets the computer work on more subjects while not slowing down, practically it means that the computer multitasks without losing performance,however the performance of the RAM is closely related to the CPU.
